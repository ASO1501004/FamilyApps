アプリ名: FamilyApps 作成者名: 川田大秀
コンセプトとアピールポイント:朝、お母さんが使うと便利なアプリです。朝、お母さんは口頭で家族全員に「今日の夕飯で食べたいもの」を訊き、それをタブレットのアプリに入力します。間違って入力した料理は項目を長くクリックすることで削除することも可能です。それをルーレットにかけて、出た結果を今日の夕飯として決定します。結果を、お母さんは夕飯として作ることになります。家族全員の食べたいものからランダムで選ばれるので、誰もが好きな食べ物を食べることができるかもしれない権利が平等に与えられることになります。また結果は家族全員にメールで送ることもできます。
